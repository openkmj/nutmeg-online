# nutmeg-online

Simple 1-on-1 battle game hitting the opponent's stone out

## Guides

Node v21.4.0!

Clone code

```
    git clone https://github.com/openkmj/nutmeg-online
```

### Server

```
    cd server
```

```
    npm install
```

```
    npm run start
```

env variables are here.

```
PORT=3001
```

### Client

```
    cd client
```

```
    npm install
```

```
    npm run dev
```

env variables are here.

```
VITE_SERVER_URL=http://localhost:3001
```
